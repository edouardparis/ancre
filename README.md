# Ancre :anchor:

![License][license-img]

A golang implementation of [OpenTimestamps](https://opentimestamps.org/) client

Ancre anchors checksums in time.

## Install

Install [the go programming language](https://golang.org).

```bash
go get github.com/ulule/ancre
go install ./...
```

## Test

```bash
go test ./...
```
